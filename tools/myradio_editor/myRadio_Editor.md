# myRadio Editor v1.0

**HU**<br>
A **myRadio Editor** egy kétnyelvű (HU/EN) Windows alkalmazás, amely megkönnyíti a rádió **stations.txt** állomáslistájának kezelését.<br>
Az alkalmazás képes letölteni az aktuális lejátszási listát a rádió **IP-címe alapján**, vagy betölteni egy meglévő listát a számítógépről.

**EN**<br>
**myRadio Editor** is a bilingual (HU/EN) Windows application that makes it easy to manage the **stations.txt** station list for your radio.<br>
The application can download the current playlist **based on the radio's IP address** or load an existing list from your computer.
