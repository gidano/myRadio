Touch preview changes:
- New board profile in Lovyan_config.h: MYRADIO_BOARD_CYD_28_ST7789_XPT2046
- New input module: src/input/input_touch.h/.cpp
- Uses PaulStoffregen/XPT2046_Touchscreen library
- Default touch pins for CYD profile:
  TOUCH_SCLK = 25
  TOUCH_MISO = 39
  TOUCH_MOSI = 32
  TOUCH_CS   = 33

Preview touch actions:
- Play screen tap: pause/play
- Play screen long press: open/close station menu
- Play screen bottom left/right: volume - / +
- Play screen top-left 64x64 area: open menu
- Menu tap on upper/lower rows: move selection
- Menu tap on middle row: select station
- Menu tap on OK line: close menu
